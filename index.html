Lenguajes FrontEnd

Html

Se utiliza para estructurar y organizar el contenido de las páginas web, definiendo elementos como encabezados, párrafos e imágenes para facilitar la presentación de información.

Css

Se utiliza para dar estilo y diseño a las páginas web, controlando la apariencia visual mediante reglas que afectan color, tamaño, posición y diseño de los elementos HTML.

JavaScript

Se utiliza para agregar interactividad y dinamismo a las páginas web, permitiendo la manipulación del contenido, respuesta a eventos del usuario y comunicación con servicios externos.
